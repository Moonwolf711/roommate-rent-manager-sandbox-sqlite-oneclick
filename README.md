# Roommate Rent Manager — Sandbox (SQLite) — One‑Click

## One‑Click Run
**Windows**: double‑click `RUN.bat`  
**macOS / Linux**:
```bash
chmod +x run.sh
./run.sh
```

This will install deps, run Prisma migrations to `prisma/dev.db`, seed sample data, and start the app at http://localhost:3000
